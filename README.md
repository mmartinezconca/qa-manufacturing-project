# QA Validation Project - Manufacturing Data System
----------------------------------------------------
## Overview

This project simulates a manifacturing data system that tracks batch production, inventory, and customer orders.  The goal is to validate data integrity and ensure business rules are followed to prevent operational errors.

## Objectives

* Validate business rules using testing cases
* Identify data quality issues
* Demonstrate QA workflow (test design --> validation)

## Project Structure 

* `data/` --> project datasets
* `docs/` --> test cases, requirements, bug reports
* `scripts/` --> SQL validation scripts

## Key Skills Demonstrated

* QA test design
* Data validation (SQL)
* Defect tracking
* Business rule analysis
